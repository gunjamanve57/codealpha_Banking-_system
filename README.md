# Bank Management System (C++)

## Description

This is a simple Bank Management System developed in C++ using Object-Oriented Programming (OOP). The project simulates basic banking operations through a menu-driven console application.

## Features

* Create Bank Account
* Deposit Money
* Withdraw Money
* Transfer Money
* View Account Details
* View Transaction History

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL Vector

## How to Run

Compile the program:

```bash
g++ src/main.cpp -o bank
```

Run the program:

```bash
./bank
```
